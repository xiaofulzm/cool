# COOL项目

>**vite+vue3**

### 项目搭建

>1. 安装vite
>
>   ```bash
>   npm  install -g create-vite-app
>   ```
>
>2. 创建vue3项目 
>
>   ```bash
>   create-vite-app Cool
>   ```
>
>3. 进入项目,安装项目依赖,运行项目
>
>   ```bash
>   cd  Cool
>   npm install
>   npm run cool
>   ```
>
>4. 安装sass vue-router
>
>   ```bash
>   npm install -D sass-loader sass (vue官方的安装方法)
>   ```
>
